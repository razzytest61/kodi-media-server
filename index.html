Kodi Media Server Guide

Welcome! This page will help you set up a media server using Kodi.

📥 Download Kodi

Click here to download Kodi for your device.

⚙️ Basic Setup

  1. Install Kodi and launch the application.
  2. Go to Settings > Services.
  3. Enable UPnP/DLNA and Web Server.
  4. Add your media sources under Videos > Files.

🌐 Access Kodi Over the Network

Enable remote control via web interface:

  • Settings > Services > Control > Allow remote control via HTTP
  • Take note of the IP address and port (default: http://:8080)
